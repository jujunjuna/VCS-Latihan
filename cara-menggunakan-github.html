1.Log in git

pastikan sudah memiliki akun git hub,masukan user name git hub menggunakan perintah $ git config --global user.name "username anda"

lalu masukan email yang terdaftar di github dengan kode : $git config --global user.email isi email anda@gmail.com

2.login github

log in kedalam website github.

3.buat repository

setelah berhasil log in di git hub ,bisa memulai membuat repository klik tombol new pada menu repositories untuk membuat repository baru

4.buat folder pada windows

fungsinya untuk menyimpan update file dari repository github yang telah anda buat

5.buka folder menggunakan git bash

buka folder lalu klik kanan dan pilih git bash here

6.ubah folder menjadi repository

setelah itu ubah folder tersebut menjadi repository menggunakan perintah $git init

7.tambahkan file ke repository

contohnya kita membuat file test.html lalu buka gitbash dan masukan perintah $git add test.html

8.buat commit

commit berfungsi menambahkan update file serta komentar masukan perintah git commit -m "first commit"

9.remote repository

berfungsi untuk mengupload file yang telah anda buat sebelumnya di local disk masukanperintah $git remote add origin git@github.com:usernamme/namarepository.git

10.push ke github

berfungsi untuk mengupload hasil akhir dari langhah langkah di atas masukan perintah git push -u origin master
